Sobre mí

Hola, soy Marcelo Rodríguez, informático, con interés en el desarrollo web y la automatización de tareas con herramientas como scraping y RPA.

Estos proyectos fueron desarrollados como parte de mi aprendizaje en Python y Web Scraping. En ellos, la información se imprime en consola, y también se guardan en archivo TXT o XLSX.

Fue una interesante oportunidad para practicar webscraping, lógica de condiciones, automatización y manejo de archivos.

Herramientas utilizadas

Herramienta Descripción
Python Lenguaje de programación usado en los scripts.
Request Permite realizar solicitudes HTTP para obtener contenido web.
BeautifulSoup Herramienta para parsear y extraer datos de HTML.
Pandas Procesa y organiza datos estructurados en tablas.
