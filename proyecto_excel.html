Scraping con Excel

Descripción técnica

Este proyecto utiliza Python y bibliotecas como Request y BeautifulSoup para extraer información de una página web.

Los datos se estructuran con Pandas y se exportan a un archivo excel usando openpyxl.

¿Cómo funciona?

  1. Se realiza petición HTTP a la URL objetivo.
  2. Se parsea el contendio HTML con BeautifulSoup.
  3. Se extraen los elementos deseados de la primera página (como títulos, precios o enlaces).
  4. Se crea un bucle while para ingresar a las otras páginas del sitio y extraer los elementos requeridos.
  5. Los datos se ordenan en un Dataframe de pandas.
  6. Finalmente, se exportan a un archivo Excel.
